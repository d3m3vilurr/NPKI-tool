# NPKI-Tool

Backup and Restore NPKI

## Installation

1. Download `NPKI_Export.bat, NPKI_Install.bat`.

---


## Usage

### Backup

```
Run 'NPKI_Export.bat' 
it will be make name of 'NPKI' folder
```


### Recover From Backup Folder

```
place NPKI folder to desktop.
Run 'NPKI_Install.bat' 
```
